# Rocky Mountain National Park — Hester Family Trip 2026

A shareable, mobile-friendly trip guide for our July 28–August 1, 2026 visit to Rocky Mountain National Park.

**🔗 Live guide:** https://jhester599.github.io/rocky-mountain-national-park-2026/index.html

## What's inside
- **Travel-day cheat card** — permit windows, booking steps, and key phone numbers on one screen
- **Reservations** — flights, rental car, and hotel confirmations
- **Day-by-day itinerary** with a recommended hike plus *Easier* / *Bigger* alternates for each day
- **Permit cheat sheet** — exactly when timed entry is required
- **Rafting & ziplining** outfitter picks
- **June 1 booking playbook** and altitude/storm/packing notes

## Itinerary at a glance
| Day | Base | Highlight |
| --- | --- | --- |
| Tue 7/28 | Estes Park | Arrive DEN, acclimate at Sprague Lake |
| Wed 7/29 | Estes → Grand Lake | Emerald Lake, then Trail Ridge Road |
| Thu 7/30 | Grand Lake | Adams Falls morning, Kremmling rafting |
| Fri 7/31 | Grand Lake → Denver | Coyote Valley, ziplining in Idaho Springs |
| Sat 8/1 | Denver | Fly home |

## ⏰ Don't miss
Timed-entry permits for July release **June 1 at 8:00 a.m. MDT** on [recreation.gov](https://www.recreation.gov/timed-entry/10086910). Book the two **Bear Lake Road** permits (Tue & Wed) first — they go fast.

## Hosting
Single self-contained `index.html` (no build step). Served via GitHub Pages: **Settings → Pages → Deploy from branch → main**.
